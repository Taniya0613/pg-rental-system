# ARDHAYA HOMES — Girls PG Website

Professional landing page for **ARDHAYA HOMES**, a Girls PG in Sector 22A, Gurgaon.

## Tech Stack

- React 18
- Vite 6
- CSS (custom properties, responsive layout)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Build for Production

```bash
npm run build
npm run preview
```

## Project Structure

```
src/
  components/
    Navbar.jsx
    Hero.jsx
    About.jsx
    RoomsPricing.jsx
    Facilities.jsx
    Gallery.jsx
    MapPlaceholder.jsx
    Contact.jsx
    Footer.jsx
    WhatsAppFloat.jsx
  App.jsx
  main.jsx
  index.css
```

## Customization

- **Google Maps**: Replace the placeholder in `MapPlaceholder.jsx` with an iframe embed from Google Maps.
- **Gallery**: Swap gradient placeholders in `Gallery.jsx` with real room photos.
- **Contact form**: Submissions open WhatsApp with a pre-filled message. Connect a backend later if needed.
